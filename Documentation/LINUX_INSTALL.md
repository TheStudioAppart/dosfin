# Linux installation

## ENVIRONMENT

- Install [`npm run env`] # Require the Terminal in fullscreen,
- Check [`npm run env:check`],
- Help [`npm run env:help`].

## DATABASE

- Create database, tables, user [`npm run db`],
- Mockup [`npm run db:mock`].
